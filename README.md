Snicker is a high performance packet sniffer written in C++

State: Testing

---

Usage:

```
    git clone https://github.com/MisterBianco/Snicker
    cd Snicker
    sudo chmod +x setup.sh && ./setup.sh
    make
    cd build
    sudo snicker -i [wireless name]
```
